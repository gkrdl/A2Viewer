# A2Viewer

아이온2 파티 뷰어 — 파티원의 직업, 닉네임, 아툴점수를 실시간 오버레이로 표시합니다.

## 기능

- 파티 멤버 자동 감지 (패킷 캡처 기반)
- 파티 신청/수락/퇴장 실시간 추적
- 캐릭터 아툴점수 자동 조회
- 오버레이 투명도 조절
- 마우스 클릭 투과 (잠금 모드)
- 전역 단축키 설정 (새로고침/숨기기)
- 아이온2 활성화 시에만 오버레이 표시 (트레이 / 설정 패널에서 변경 가능)

## 사용법

1. **A2Viewer-1.0.0.exe** 실행
2. 아이온2 게임 실행
3. 파티에 참가하면 자동으로 파티원 정보 표시

## 단축키

| 기능 | 기본 키 | 변경 |
|------|---------|------|
| 숨기기/보이기 | `Alt + \`` | 설정(톱니바퀴) |
| 목록 새로고침 | `Ctrl + R` | 설정(톱니바퀴) |

## 설정

타이틀바의 톱니바퀴 아이콘 또는 트레이 우클릭 메뉴에서 설정 가능:

- **아이온2 활성화 시에만 표시** — 게임 창이 포그라운드일 때만 오버레이 표시 (트레이 메뉴와 연동)
- **단축키 변경** — 새로고침, 숨기기/보이기 키를 자유롭게 설정

## 요구사항

- Windows 10 이상
- [Npcap](https://npcap.com/) 설치 필요 (패킷 캡처용)
  - 설치 시 **WinPcap API-compatible Mode** 체크

## 트레이 아이콘

- 좌클릭: 오버레이 열기
- 우클릭: 메뉴 (아이온2 활성화 시에만 표시 / 종료)
