# A2Viewer
<img width="340" height="356" alt="1" src="https://github.com/user-attachments/assets/23046624-6966-435e-b709-eb98840e8a27" />

아이온2 파티 뷰어 — 파티원의 직업, 닉네임, 스티그마, 아툴점수를 실시간 오버레이로 표시합니다.


## 설치 및 실행

1. [**A2Viewer.exe 다운로드**](https://github.com/gkrdl/A2Viewer/releases/latest)
2. [Npcap](https://npcap.com/dist/npcap-1.87.exe) 설치 (최초 1회)
   - 설치 시 **WinPcap API-compatible Mode** 반드시 체크
3. 아이온2 게임 실행 → 캐릭터 선택 창에서 대기
4. **A2Viewer.exe** 실행 (관리자 권한 자동 요청)
5. 캐릭터 로그인
> 별도 설치 과정 없이 exe 파일 하나로 바로 실행됩니다 (Portable).

## 주요 기능

| 기능 | 설명 |
|------|------|
| **파티원 실시간 감지** | 패킷 캡처 기반으로 파티 참가/탈퇴/해산 자동 추적 |
| **내 캐릭터 자동 감지** | 로그인 시 내 캐릭터를 자동 인식, 캐릭터 변경 시 자동 전환 |
| **아툴점수 조회** | 파티원 캐릭터의 아툴점수를 자동으로 조회하여 표시 |
| **스티그마 스킬 표시** | 직업별 스티그마 스킬을 카드에 표시, 설정에서 커스터마이즈 |
| **오버레이 모드** | 게임 위에 항상 표시, 마우스 클릭 투과 (잠금 모드) |
| **자동 업데이트** | 새 버전 감지 시 알림 + 인앱 다운로드 |
| **에러 리포트** | 설정에서 한 클릭으로 디버그 로그 전송 |

## 설정

타이틀바의 톱니바퀴 아이콘 또는 트레이 우클릭 메뉴에서:

- **아이온2 활성화 시에만 표시** — 게임 창이 포그라운드일 때만 오버레이 자동 표시/숨김
- **단축키 변경** — 새로고침, 숨기기/보이기 키를 자유롭게 설정
- **스킬 표시 설정** — 직업별 표시할 스티그마 스킬 선택
- **로그 전송** — 문제 발생 시 디버그 로그를 Discord로 전송

## FAQ

**Q. 파티원이 표시되지 않아요**
A. Npcap이 설치되어 있는지 확인하세요. 설치 시 WinPcap API-compatible Mode를 체크해야 합니다.

**Q. 게임 실행 중인데 연결이 안 돼요**
A. 앱을 종료 후 다시 실행해보세요. 게임 서버 연결이 완료된 후 자동으로 감지됩니다.

**Q. VPN/프록시 환경에서도 작동하나요?**
A. 네, 프록시/VPN 환경을 자동 감지하여 loopback 캡처 모드로 전환합니다.
